# Criador-de-musicas

Como acessar:
Clique no botão "Publish" no painel de gerenciamento do Manus (canto superior direito)
Você receberá um link público (algo como https://compositor-ia.manus.space )
Abra o link no navegador e comece a criar letras musicais
Requisitos para usar:
Navegador moderno (Chrome, Firefox, Safari, Edge)
Conexão com internet
Conta Manus (para login e salvar suas composições)
Se quiser rodar localmente (desenvolvimento):
Se você quiser modificar o código ou rodar na sua máquina, precisaria:
Node.js 22+ e pnpm
Clonar o repositório e rodar pnpm install && pnpm dev
Configurar variáveis de ambiente (chaves de API)
Mas para usar a aplicação normalmente, basta o navegador. Tudo está pronto na nuvem! 🚀
