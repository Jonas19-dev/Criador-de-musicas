O projeto consiste no desenvolvimento de um site interativo para criação assistida de letras musicais utilizando Inteligência Artificial e técnicas de Engenharia de Prompt. A proposta da aplicação é oferecer uma plataforma web moderna, funcional e responsiva, capaz de atuar como um compositor virtual inteligente, auxiliando usuários na criação de músicas originais de maneira criativa, dinâmica e intuitiva.

O site deverá funcionar diretamente no navegador, permitindo que o usuário informe parâmetros específicos para geração da música, como tema, gênero musical, humor desejado e palavras-chave obrigatórias. A partir dessas informações, a aplicação deverá utilizar modelos de linguagem baseados em IA (LLMs) para gerar letras musicais coerentes, emocionalmente envolventes e organizadas estruturalmente.

A interface da aplicação deverá ser intuitiva e amigável, contendo formulários para entrada dos dados, botões de geração e refinamento das músicas, além de uma área dedicada à exibição das letras produzidas pela Inteligência Artificial. O sistema deverá permitir interação contínua com a IA, possibilitando novas iterações, continuações da música, reescrita de versos específicos, criação de novos refrões e adaptação da composição para diferentes estilos musicais.

As letras geradas deverão seguir uma estrutura clara e organizada, utilizando divisões tradicionais como Verso 1, Pré-Refrão, Refrão, Verso 2, Ponte e Refrão Final. O refrão deverá ser marcante e memorável, transmitindo a principal emoção da composição. Os versos deverão desenvolver a narrativa da música de forma consistente, utilizando metáforas, figuras de linguagem e imagens poéticas compatíveis com o gênero e humor definidos pelo usuário. O sistema deverá evitar repetições excessivas e garantir que as palavras-chave fornecidas sejam incorporadas naturalmente à letra.

O comportamento da geração textual deverá variar conforme o gênero musical selecionado. Em músicas de rap e hip-hop, a IA deverá produzir rimas fortes, flow marcante e linguagem impactante. Em músicas pop, o foco deverá estar em refrões comerciais e fáceis de memorizar. No rock, a composição deverá transmitir intensidade emocional e imagens fortes. Em músicas eletrônicas, o texto poderá apresentar características atmosféricas, sensoriais e estrategicamente repetitivas. Já no sertanejo, a composição deverá enfatizar emoções, relacionamentos e linguagem popular. Em estilos como trap, a escrita deverá incorporar elementos urbanos, atitude e estética contemporânea.

A aplicação deverá utilizar técnicas de Engenharia de Prompt para controlar e melhorar a qualidade das respostas geradas pela IA. O sistema deverá possuir um Prompt Principal de Geração responsável por criar a letra base da música a partir dos parâmetros fornecidos pelo usuário. O prompt deverá instruir a IA a atuar como um compositor profissional especializado no gênero musical selecionado, criando uma letra original baseada no tema, humor e palavras-chave definidos pelo usuário, organizando a composição em versos, refrão e ponte de maneira estruturada.

Além do prompt principal, o sistema deverá implementar prompts iterativos e secundários para funcionalidades de refinamento, permitindo que o usuário solicite alterações específicas, como reescrita de versos, criação de pontes, geração de novos refrões, continuação da música ou adaptação para outro gênero musical. O projeto também deverá documentar as estratégias de Engenharia de Prompt utilizadas, incluindo técnicas como zero-shot prompting, persona-based prompting, chain-of-thought prompting, contextual prompting e few-shot prompting, quando aplicável.

O desenvolvimento do sistema deverá seguir uma arquitetura moderna dividida entre front-end e back-end. No front-end, recomenda-se a utilização de React.js, Vue.js ou Angular para criação de uma interface moderna, reativa e responsiva. O front-end será responsável por capturar as entradas do usuário, enviar requisições ao servidor e exibir as letras geradas pela IA de forma organizada e visualmente agradável.

No back-end, recomenda-se a utilização de Node.js com Express.js, permitindo o uso de JavaScript ou TypeScript em toda a aplicação. O servidor back-end deverá atuar como intermediário entre o front-end e a API do modelo de linguagem, sendo responsável por receber os dados enviados pelo usuário, construir dinamicamente os prompts, realizar as chamadas para a API do LLM, processar as respostas geradas pela IA e retornar os resultados formatados ao front-end, além de garantir a segurança da chave da API. Como alternativa, poderá ser utilizado Python com Flask ou FastAPI, especialmente pela forte integração de Python com aplicações de Inteligência Artificial.

A aplicação deverá integrar APIs de modelos de linguagem como OpenAI GPT, Google Gemini ou Anthropic Claude, utilizando bibliotecas oficiais como openai-node, openai para Python ou SDKs equivalentes.

Como resultado final, os alunos deverão entregar um site funcional e responsivo, hospedado online ou executado localmente com instruções claras de instalação e execução. A entrega deverá incluir uma interface interativa para geração e refinamento de letras musicais, um documento detalhado contendo todos os prompts utilizados e as estratégias de Engenharia de Prompt aplicadas, além do código-fonte completo do projeto, incluindo front-end, back-end e integração com APIs de IA.

Também deverão ser apresentadas evidências do funcionamento da aplicação, como capturas de tela, vídeos demonstrativos ou link da aplicação hospedada. O projeto deverá ser disponibilizado em um repositório Git organizado, como GitHub, GitLab ou Bitbucket, contendo um README.md detalhado com instruções de instalação, configuração do ambiente, execução do projeto e descrição das tecnologias utilizadas.

O sistema deverá priorizar criatividade, organização textual, experiência do usuário e integração eficiente com modelos de Inteligência Artificial
